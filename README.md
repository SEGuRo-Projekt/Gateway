# Gateway
Scripts for the measurement data transmission to the SEGuRo platform

## Installation

```shell
pip install .
```

## Development

```shell
pip install -e .
```


## MQTT Topics

`measurements/<loc_id>/<measuring_device_id>/<measuring_point_id>`
